# CurrencyConverter

Console currency converter written in C++ with exchange rates, error handling and conversion history.

## Features

- Convert USD, EUR, GBP to RUB
- View all exchange rates
- Error handling for unknown currencies and invalid amounts
- Save conversion history to file

## Technologies

- C++
- STL (map, vector)
- Exception handling (try/catch, throw)
- File I/O (fstream)

## How to run

1. Clone the repository
2. Open `CurrencyConverter.cpp` in Visual Studio or any C++ compiler
3. Build and run
4. Conversion history is saved to `history.txt`
